## Design Guidelines

- **File format:** JPG or PNG
- **Aspect ratio:** 1.91:1 (1200x630)
- **File size:** Under 300KB for faster loading
- **Text:** Large, readable text (minimum 40px font size)
- **Safe zone:** Keep important content within center 1200x600 area
- **Brand consistency:** Use Green Laniel color palette and fonts

## Testing

Test OG images using:
- **Facebook Sharing Debugger:** https://developers.facebook.com/tools/debug/
- **Twitter Card Validator:** https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector:** https://www.linkedin.com/post-inspector/
